# 📌 Overview

Climate Guardian AI is a modern AI-powered climate intelligence platform that combines real-time weather information, air quality monitoring, climate risk analysis, and AI-generated recommendations into one beautiful dashboard.

It helps users understand environmental conditions and supports smarter, data-driven decisions.

- 🌍 Real-time Weather Information

# ⚙️ Installation

Clone Repository

```bash
git clone https://github.com/vinaysingh-05/Climate-Guardian-AI.git
```

Open Folder

```bash
cd Climate-Guardian-AI
```

Install Dependencies

```bash
npm install
```

Create Environment File

```env
VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
VITE_WEATHER_API_KEY=YOUR_WEATHER_API_KEY
VITE_GOOGLE_MAPS_API_KEY=YOUR_GOOGLE_MAPS_API_KEY
```

Run Development Server

```bash
npm run dev
```

Build Project

```bash
npm run build
```
